Synchronization

Objective

This step fetches the environment from the RVX server.

  • You must run this once when receiving the Git repository for the first time.

  • After the first run, it will be automatically triggered during the update process.

  • During synchronization, you will be prompted to enter your RVX server account credentials.

  • Make sure you know your account credentials in advance.

  • These credentials will be saved in ${RVX_MINI_HOME}/.rvx_server_config.

  • If you need to change your RVX account info, simply delete that file.

Instruction @ Any OS

cmd) cd ${RVX_MINI_HOME}
cmd) make sync
> When prompted for SSH access, answer ‘no’

(Force) Resynchronization

Objective

Run this if an issue occurs during the sync process.

Instruction @ Any OS

cmd) cd ${RVX_MINI_HOME}
cmd) make resync
