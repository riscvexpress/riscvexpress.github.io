RISC-V eXpress Homepage

Overview

RISC-V eXpress (RVX) is a hardware/software development kit
for designing low-power SoCs for embedded systems,
such as MCUs or lightweight application processors.

Developed by ETRI, South Korea, with support from the
ICT R&D Program of MSIT/IITP (No. 2018-0-00197).

Key Features

  • Compatible with both Linux and Windows
  • Free for non-commercial research use with citation
  • Designed for ease of use
  • Extensible with AMBA-compliant IP blocks
  • Integrated with an SNN learning framework
