Overview

  • This manual is intended for developing software for SoCs using RVX.

  • All software must be developed with this constraint in mind to ensure correct functionality since the RVX SoC is designed as a lightweight platform and therefore does not support a general-purpose operating system.

Things to Know

  • Manuals are available online - riscvexpress.github.io

  • Any part starting with # should be replaced or modified according to your environment.

  • On Linux, use the bash shell for command-line operations.

  • On Windows, use the Windows Power Shell for command-line operations.
